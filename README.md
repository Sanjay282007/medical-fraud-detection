# 🏥 MedGuard AI – Healthcare Insurance Fraud Prevention & Investigation Platform

## Overview

MedGuard AI is a Healthcare Insurance Fraud Prevention and Investigation Platform designed to help insurance providers identify suspicious medical claims, assess fraud risk, investigate high-risk cases, and generate actionable insights through analytics and risk intelligence.

The platform combines Machine Learning, Risk Scoring, Analytics, and Investigation Workflows to support faster and more accurate fraud detection.

## Project Modules

### 1. Authentication Module

Handles user registration, login, logout, and access control.

### 2. Fraud Detection Module

Predicts fraudulent claims using machine learning models.

### 3. Risk Assessment Module

Calculates risk scores and categorizes claims into risk levels.

### 4. Investigation Module

Supports review and verification of suspicious claims.

### 5. Analytics Module

Provides visual insights into fraud patterns and claim activity.

### 6. Reporting Module

Generates downloadable PDF reports for investigations and audits.

## Installation

### Clone Repository

```bash
git clone https://github.com/Sanjay282007/medical-fraud-detection.git
cd medical-fraud-detection
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
